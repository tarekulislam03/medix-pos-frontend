# Medix Pos - Frontend 

## Overview

Frontend layer handling UI, client-side logic, and backend communication. Built for scalability, clean structure, and responsive performance.

---

## Structure

* `assets/` – static files
* `src/` – main code (components, screens, services, utils)
* `App.js` – root component
* `index.js` – entry point
* `.env` – environment variables

---

## Key Points

* Component-based architecture
* Centralized API handling
* Optimized UI rendering
